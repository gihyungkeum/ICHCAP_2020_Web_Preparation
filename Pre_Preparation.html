Atom 에디터 실행방법

  • 터미널(커맨드)창 열기: (window) + R --> cmd --> atom .
  • 터미널(커맨드창) 기초 명령어 : mkdir cd cd..
  • 터미널 환경(터미널 창)에서 윈도우 환경(로컬드라이브/내 pc 환경) 열기 : (window) + e
  • 아톰 터미널 환경과 윈도우 환경 연도 : mkdir을 통해 폴더 생성
  • 윈도우 환경에서 우클릭 --> 새로만들기 --> 텍스트 파일생성 : 아톰에디터에도 연동되어 파일생성
  • 아톰 에디터에서 바로 파일생성하여 실행 : >atom 파일명.html --> 엔터
  • 터미널 창이 작동 안 될 경우-->위도우 환경에 우클릭 : open with atom

    • Atom 에디터 몇가지 단축키 tips

    1. 주석처리
    2. Ctrl + / : HTML CSS /* */
    3. 여러줄선택 : shift + 방향키 + /
    4. 여러줄 indentation : Ctrl + [ 또는 ]
    5. 화면크기 조절 : Ctrl + 또는 -
    6. 화면분할 : 윈도우 + 방향키(< ^ > )
